# Einkaufsliste

This is a Mathematisch-technischer Softwareentwickler(MATSE25) project designed to help students become efficient in the Java programming language during the first semester of the Bachelor Studies.

The team members are working on different assignments, such as:
Database
User Interface

Tools Used
The team members work with the following tools:
Visual Studio Code
Eclipse

Main Features of the Software
Users can add items to a shopping list
Users can delete items from the shopping list
Users can modify items in the shopping list
